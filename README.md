# وب‌سایت لبنیات صباح

صفحهٔ اصلی مدرن، فارسی و کاملاً RTL برای برند لبنیات صباح؛ ساخته‌شده با Next.js، TypeScript، Tailwind CSS و Framer Motion.

## نسخه آنلاین

[مشاهده وب‌سایت صباح](https://moazzamitr-bit.github.io/sabah/)

## قابلیت‌ها

- هیروی سینمایی با پارالاکس اسکرول و لایه‌های متحرک
- هدر واکنش‌گرا با حالت فشرده هنگام اسکرول
- ورود مرحله‌ای کارت‌های محصولات
- زنجیرهٔ تولید با پیشرفت وابسته به اسکرول
- منوی موبایل و جستجوی متحرک
- پشتیبانی از `prefers-reduced-motion`
- تصاویر اختصاصی محصولات و هویت بصری صباح

## اجرا

```bash
npm install
npm run dev
```

سپس [http://localhost:3000](http://localhost:3000) را باز کنید.

## بررسی تولید

```bash
npm run typecheck
npm run build
```
